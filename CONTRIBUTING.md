# Contributing to Clario

Thank you for your interest in contributing to **Clario** — the crypto-native tutoring marketplace where tutors and students transact with crypto, build reputation on-chain via EQBSL, and learn instantly.

## Areas Where Contributions Are Welcome

- **Frontend Components** — UI improvements, new pages, accessibility enhancements, and responsive design refinements using React and Tailwind CSS.
- **EQBSL Integration** — Deeper integration with the [EQBSL](https://github.com/Steake/EQBSL) trust-as-ledgered-state framework, including reputation scoring, trust parameter governance, and settlement layer improvements.
- **Documentation** — Improving or expanding the README, adding inline code comments, and writing guides for new contributors.
- **Testing** — Adding unit tests, integration tests, and end-to-end test coverage for existing and new features.

## How to Contribute

1. **Fork** the repository on GitHub.
2. **Create a branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**, keeping commits focused and descriptive.
4. **Push** your branch to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
5. **Open a Pull Request** against the `main` branch of this repository, describing what you changed and why.

All contributions are welcome — please keep pull requests focused and clearly described. We review all submissions and aim to provide timely feedback.
